# Random-Quote-Generator
**Автор:** [Фризе Ольга]
**Дата** 04.05.2026
## Описание

Random Quote Generator — это графическое приложение на Python для генерации случайных цитат с возможностью сохранения истории, фильтрации и добавления собственных цитат.

Приложение предоставляет следующие функции:

* генерация случайных цитат из предопределённого списка;
* отображение полной информации о цитате (текст, автор, тема);
* сохранение истории всех сгенерированных цитат;
* фильтрация истории по автору и по теме;
* добавление новых цитат в базу данных;
* сохранение всех данных между запусками приложения (в JSON файлах).

**Используемые технологии:**
* Python 3.6+;
* библиотека `tkinter` для графического интерфейса;
* библиотека `requests` не требуется — данные хранятся локально;
* формат хранения данных — JSON.

## Требования

* Python 3.6 или выше;
* стандартная библиотека `tkinter` (обычно входит в поставку Python);
* библиотека `json` (встроена в Python);
* библиотека `random` (встроена в Python).


## Особенности реализации

1. **Локальное хранение данных**
   * все данные хранятся в JSON файлах на стороне пользователя (`predefined_quotes.json` и `quote_history.json`);
   * не требуется подключение к интернету или внешний сервер;
   * данные сохраняются между запусками приложения.

2. **Гибкость базы цитат**
   * предопределённый список цитат загружается при запуске;
   * пользователи могут добавлять собственные цитаты через интерфейс;
   * новые цитаты сразу становятся доступны для генерации.

3. **Механизм генерации**
   * используется модуль `random.choice()` для случайного выбора цитаты из списка;
   * каждая сгенерированная цитата автоматически добавляется в историю.

4. **Система фильтрации**
   * реализована нечёткая фильтрация (поиск подстроки в поле автора/темы);
   * регистронезависимый поиск (`lower()`);
   * результаты отображаются в отдельном списке для удобства сравнения.

5. **Обработка ошибок**
   * проверка на пустые поля при добавлении новых цитат;
   * обработка ошибок чтения/записи JSON файлов;
   * уведомления об ошибках через `messagebox.showerror()`.

6. **Интерфейс пользователя**
   * разделение на функциональные блоки (генератор, добавление, фильтрация, история);
   * интуитивное расположение элементов управления;
   * обратная связь через диалоговые окна (`messagebox`).

7. **Сохранение состояния**
   * история цитат сохраняется после каждого добавления новой записи;
   * при запуске приложения автоматически загружаются:
     * предопределённые цитаты (или создаётся файл с дефолтными);
     * история ранее сгенерированных цитат.

8. **Структура данных**
   * цитаты хранятся как словари с полями: `text`, `author`, `topic`;
   * история — массив таких словарей;
   * JSON обеспечивает читаемость и лёгкость редактирования вручную.

9. **Адаптивность интерфейса**
   * элементы растягиваются при изменении размера окна;
   * корректное отображение длинных текстов (перенос в текстовом поле);
   * использование `ttk` для современного вида виджетов.

10. **Валидация ввода**
    * проверка заполнения всех полей при добавлении новой цитаты;
    * очистка полей после успешного добавления;
    * подсказки через сообщения об ошибках.

## Требования

* библиотека `json` (встроена в Python);
